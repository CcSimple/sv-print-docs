              温馨提示

              目前国内项目使用 Svelte 开发管理系统的可能比较少。所以这里以 Vue 使用方式作为示例。

              # 安装

              • Node.js v16+open in new window(开发环境是 16.18.1)
              • Vite v3+open in new window(开发环境是 ^3.2.5)
              • Webpack v4+open in new window(使用 vue create 创建的项目测试可正常引入)
              • jQuery/纯js 项目支持open in new window(使用 html + js 项目测试可正常引入 (需要demo 联系我))

              目前支持框架/js 如下：

              PackageStatusDescription
              sv-printopen in new windowopen in new windowSvelte/Vanilla JS 组件
              @sv-print/reactopen in new windowopen in new windowReact 组件
              @sv-print/vueopen in new windowopen in new windowVue 2 组件
              @sv-print/vue3open in new windowopen in new windowVue 3 组件
              @sv-print/hiprintopen in new windowopen in new windowhiprint core (js 库)

              请选择你需要的版本

              npm i @sv-print/vue

              # 引入打印样式

              重要提醒

              需要复制【node_modules/@sv-print/hiprint/dist/print-lock.css】到开发资源目录。
              例如: Vue 项目的 public 目录。
              假如你部署的网站是: https://www.abcd.com/index.html 那么确保 https://www.abcd.com/print-lock.css 能够正常访问

              # 引入组件样式

              // 一般在入口文件(index/main)添加
import "sv-print/dist/style.css";

              # 引入组件

              <template>
  <Designer :template="template" :printData="printData" @onDesigned="onDesigned">
    <!-- 自定义 header -->
    <div class="header" slot="header">
      <div>header slot test1</div>
    </div>
    <!-- 自定义 左侧拖拽 元素 -->
    <div slot="draggableEls">
      <!-- 注意这里 tid="defaultModule.text" 如果是自定义 provider 的内容, 也需要先传对应的 provider -->
      <div class="ep-draggable-item" tid="defaultModule.text">
        <i class="iconfont sv-text" />
        <p>-文本-</p>
      </div>
    </div>
  </Designer>
</template>

<script>
import { Designer } from "@sv-print/vue";

export default {
  components: { Designer },
  data() {
    return { template: {}, printData: { name: "abcde" } };
  },
  methods: {
    onDesigned(e) {
      // 这里会 回调 2 个 关键对象
      console.log(e);
      console.log(e.hiprint); // hiprint 模块
      // 更多 API 可查看 log
      console.log(e.designerUtils); // 设计器 对象
      console.log(e.designerUtils.printTemplate); // 模板 对象
    },
  },
};
</script>
