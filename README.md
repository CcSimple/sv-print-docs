# sv-print 文档源码

sv-print 是一个使用 Svelte 构建的打印设计器组件（[基于 hiprint](http://hiprint.io)）。它也可以用于其他 UI 库/框架，如 React、Vue 和 Angular。

本仓库将编写 sv-print 使用文档，以及 hiprint 使用文档

> 感兴趣的小伙伴可以 fork 后编写贡献文档啦~
